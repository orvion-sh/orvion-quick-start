Server-Side Code

Python (FastAPI)
from orvion.fastapi import require_payment

@app.get("/api/premium")
@require_payment(
    amount="0.01",
    currency="USDC",
)
async def premium(request):
    return {"data": "premium"}
Node.js (Express)
const { requirePayment } = require('orvion');

app.get('/api/premium', 
    requirePayment({
        amount: '0.01',
        currency: 'USDC',
    }),
    (req, res) => {
        res.json({ data: 'premium' });
    }
);
